SQL(Structured Query Language): is a programming language that is used to communicate with a relational databse.

Relational database: in the database, datas are structured into tables, tables are related to each other based on common attributes.Each tabel has a unique name, and in each table,each row is a record,while each column represents an attribute(a field).

Each column has a name and a data type, there are 7 kinds of data types:

  • Interger: used for numbers without decimal points(like:1,99)
  • Decimal and Float： used for fractional numbers(for instince:2.0,3.69).The difference between them is,decimal is used for precise decimal arithmetic,for example:financial calculations,currency representation.Float is used for approximate arithmetic with real numbers.
  • Char and Varchar: used for characters and strings.CHAR for fixed-length strings,and VARCHAR is for variable-length strings
  • Date and Time: used for date and time
