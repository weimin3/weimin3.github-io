SQL(Structured Query Language): is a programming language that is used to communicate with a relational databse.

Relational database: in the database, datas are structured into tables, tables are related to each other based on common attributes.Each tabel has a unique name, and in each table,each row is a record,while each column represents an attribute(a field).

Each column has a name and a data type, there are 7 kinds of data types:

  • Interger: used for numbers without decimal points(like:1,99)
  • Decimal and Float: used for fractional numbers(for instince:2.0,3.69).The difference between them is,decimal is used for precise decimal arithmetic,for example:financial calculations,currency representation.Float is used for approximate arithmetic with real numbers.
  • Char and Varchar: used for characters and strings.CHAR for fixed-length strings,and VARCHAR is for variable-length strings
  • Date and Time: used for date and time

SQL constraintsare rules applied to columns in a table to ensure the accuracy and reliability of the data within the database.

  • PRIMARY KEY:Each table can have only one primary key,the primary key column(s) cannot contain NULL values. it uniquely identifies each record in a table.
  • FOREIGN KEY: Ensures that the value in one table matches a value in another table,helps maintain referential integrity. 
    
      ```sql
      CREATE TABLE departments (
          department_id INT PRIMARY KEY,
          department_name VARCHAR(100)
      );

      CREATE TABLE employees (
          employee_id INT PRIMARY KEY,
          name VARCHAR(100),
          department_id INT,
          FOREIGN KEY (department_id) REFERENCES departments(department_id)
      );
```
  • UNIQUE:Ensures that all values in a column are unique.A table can have multiple unique constraints. 
    
      CREATE TABLE users (
        user_id INT PRIMARY KEY,
        email VARCHAR(255) UNIQUE
      );
  • NOT NULL:Ensures that a column cannot have NULL values. 
    
      CREATE TABLE products (
        product_id INT PRIMARY KEY,
        product_name VARCHAR(100) NOT NULL
      );
  • DEFAULT:Provides a default value for a column when none is specified. 
    
        CREATE TABLE orders (
          order_id INT PRIMARY KEY,
          order_date DATE DEFAULT CURRENT_DATE
      );
  • CHECK:Ensures that the value in a column meets a specific condition. 
    
      CREATE TABLE employees (
          employee_id INT PRIMARY KEY,
          salary DECIMAL(10, 2),
          CHECK (salary > 0)
      );
  • AUTO_INCREMENT (MySQL-specific):Typically used for primary key columns,automatically generates a unique value for each new record. 
    
        CREATE TABLE employees (
            employee_id INT AUTO_INCREMENT PRIMARY KEY,
            name VARCHAR(100)
        );
